--- title: "Installing Yt" h2: "Installing Yt" ---

Add Yt to any bundled Ruby project by adding the line gem 'yt', '~> {{ page.version }}' to its Gemfile.

You can also install Yt in your system by opening the Terminal and typing gem install yt.

If you need to update version, type gem update yt, or run bundle update if inside a Ruby project.

Once Yt is installed, you can open the Terminal and type:

{% highlight sh %} yt info {% endhighlight %}

This will print out the version number and the annotation fetched from this YouTube video.

If you see the following message, congratulations! You are ready to start the tutorial.

{% highlight sh %} Yt version {{ page.version }} Hello, world! {% endhighlight %}
